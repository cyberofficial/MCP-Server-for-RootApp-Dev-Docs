<h1>Manifest permissions | Root developer documentation</h1>
<header><h1>Manifest <code>permissions</code></h1></header>
<p>Permissions control which parts of the Root Community API your code is able to use.</p>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="details">Details<a href="#details" class="hash-link" aria-label="Direct link to Details" title="Direct link to Details" translate="no">​</a></h2>
<p>Declare your permissions in the <code>permissions</code> section of your manifest.</p>
<table><thead><tr><th></th><th></th></tr></thead><tbody><tr><td><strong>Name</strong></td><td><code>permissions</code></td></tr><tr><td><strong>Type</strong></td><td><code>object</code></td></tr><tr><td><strong>Required</strong></td><td>No</td></tr></tbody></table>
<p>All permissions default to <code>false</code>; set the ones you need to <code>true</code>.</p>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="which-permissions-are-available">Which permissions are available?<a href="#which-permissions-are-available" class="hash-link" aria-label="Direct link to Which permissions are available?" title="Direct link to Which permissions are available?" translate="no">​</a></h2>
