<header><h1>Upload your code</h1></header>
<p>The <code>rootsdk upload</code> command deploys your code to the Root cloud. You must provide the package file path and a valid authentication token. You get your authentication token from <a href="https://dev.rootapp.com" target="_blank" rel="noopener noreferrer" class="">Root Developer Portal</a>.</p>
