<h1>JobScheduler | Root developer documentation</h1>
<header><h1>JobScheduler</h1></header><blockquote>
<p><strong>JobScheduler</strong>: <a class="" href="/docs/bot-api-reference/type-aliases/TypedEventEmitter/"><code>TypedEventEmitter</code></a>&lt;<a class="" href="/docs/bot-api-reference/type-aliases/JobScheduleEvents/"><code>JobScheduleEvents</code></a>&gt; &amp; <code>object</code></p>
</blockquote>
<p>Represents a job scheduling system that notifies you when it's time for one of your tasks to run. The job scheduler doesn't run your tasks, it notifies you via an event at the scheduled time so you can run the task yourself.</p>

<p>Creates a new job.</p>

<h4 class="anchor anchorTargetStickyNavbar_Vzrq" id="returns">Returns<a href="#returns" class="hash-link" aria-label="Direct link to Returns" title="Direct link to Returns" translate="no">​</a></h4>
<p><code>Promise</code>&lt;<a class="" href="/docs/bot-api-reference/type-aliases/JobRecord/"><code>JobRecord</code></a>&gt;</p>
<p>A promise that resolves with the created job record. The returned object contains all the same values you passed in the create request with one difference: it adds the <code>jobScheduleId</code> that the <code>JobScheduler</code> created to uniquely identify this job. You'll need the <code>jobScheduleId</code> to uniquely identify this job in edit or delete operations.</p>

<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="delete">delete()<a href="#delete" class="hash-link" aria-label="Direct link to delete()" title="Direct link to delete()" translate="no">​</a></h3>
<p>Deletes a job by its ID. The <code>jobScheduleId</code> will be unique, so this method will delete at most one job.</p>
<h4 class="anchor anchorTargetStickyNavbar_Vzrq" id="parameters-1">Parameters<a href="#parameters-1" class="hash-link" aria-label="Direct link to Parameters" title="Direct link to Parameters" translate="no">​</a></h4>

<h4 class="anchor anchorTargetStickyNavbar_Vzrq" id="returns-1">Returns<a href="#returns-1" class="hash-link" aria-label="Direct link to Returns" title="Direct link to Returns" translate="no">​</a></h4>
<p><code>Promise</code>&lt;<code>void</code>&gt;</p>
<p>A promise that resolves when the job is deleted. It's not considered an error if the parameter you pass doesn't match any jobs. If needed, you can use <code>get</code> or <code>list</code> to determine whether your parameters will match an existing job.</p>
