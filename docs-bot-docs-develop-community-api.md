<h1>Community access | Root developer documentation</h1>
<header><h1>Community access</h1></header>
<p>The <em>community access</em> section of the Root API is a set of methods and events that let your server code interact with the community in which you're running.</p>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="what-does-community-access-enable">What does community access enable?<a href="#what-does-community-access-enable" class="hash-link" aria-label="Direct link to What does community access enable?" title="Direct link to What does community access enable?" translate="no">​</a></h2>
<p>The community API lets you build a wide range of experiences. Here are a few common categories with some ideas.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="member-management">Member management<a href="#member-management" class="hash-link" aria-label="Direct link to Member management" title="Direct link to Member management" translate="no">​</a></h3>
<p>Track member activity and automatically promote users when they hit milestones. Set up a smooth onboarding flow with a checklist, starter role, and welcome message. Add an XP system that rewards participation and unlocks new roles or badges as members level up.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="events-and-coordination">Events and coordination<a href="#events-and-coordination" class="hash-link" aria-label="Direct link to Events and coordination" title="Direct link to Events and coordination" translate="no">​</a></h3>
<p>Post reminders for upcoming events and tag the right roles automatically. Let members RSVP with reactions or commands, and log attendance for future rewards. Keep engagement high by posting daily challenges or prompts.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="messaging-and-content">Messaging and content<a href="#messaging-and-content" class="hash-link" aria-label="Direct link to Messaging and content" title="Direct link to Messaging and content" translate="no">​</a></h3>
<p>Create temporary channels on a schedule and clean them up when they’re no longer needed. Pin popular messages based on reactions or keywords, and unpin older ones. Enable quick polls in chat and use emoji reactions to track results.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="moderation-and-safety">Moderation and safety<a href="#moderation-and-safety" class="hash-link" aria-label="Direct link to Moderation and safety" title="Direct link to Moderation and safety" translate="no">​</a></h3>
<p>Auto-delete rule-breaking messages and issue warnings or kicks based on filters. Track repeat offenses and auto-ban users after a set number of strikes. Flag suspicious join activity to catch bot raids or alt floods.</p>

<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="which-operations-are-available">Which operations are available?<a href="#which-operations-are-available" class="hash-link" aria-label="Direct link to Which operations are available?" title="Direct link to Which operations are available?" translate="no">​</a></h2>
<p>Think of your <strong>server like a community member</strong>. Your code can do almost everything a human can in the community.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="what-your-code-can-do">What your code can do<a href="#what-your-code-can-do" class="hash-link" aria-label="Direct link to What your code can do" title="Direct link to What your code can do" translate="no">​</a></h3>
<ul>
<li class="">Create channels and channel groups</li>
<li class="">Post messages in channels</li>
<li class="">Upload files to channels</li>
<li class="">Pin messages</li>
<li class="">Kick or ban members</li>
<li class="">Create new roles</li>
<li class="">Assign roles to members</li>
<li class="">And more...</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="events-your-code-can-watch-for">Events your code can watch for<a href="#events-your-code-can-watch-for" class="hash-link" aria-label="Direct link to Events your code can watch for" title="Direct link to Events your code can watch for" translate="no">​</a></h3>
<ul>
<li class="">Messages are created, edited, deleted, or pinned</li>
<li class="">Members join or leave the community</li>
<li class="">Roles are created, updated, or deleted</li>
<li class="">The community name or picture changes</li>
<li class="">And more...</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="what-your-code-cant-do">What your code <strong>can’t</strong> do<a href="#what-your-code-cant-do" class="hash-link" aria-label="Direct link to what-your-code-cant-do" title="Direct link to what-your-code-cant-do" translate="no">​</a></h3>
<ul>
<li class="">Cannot send notifications</li>
<li class="">Cannot respond to community join invites</li>
<li class="">Cannot interact with <em>Friends</em> lists</li>
</ul>
