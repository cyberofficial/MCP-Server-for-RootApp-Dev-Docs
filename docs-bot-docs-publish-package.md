<header><h1>Package your code</h1></header>
<p>The <code>rootsdk build package</code> command packages your compiled code for distribution. It creates a <code>.pkg</code> file based on your <code>root-manifest.json</code>.</p>

<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="options">Options<a href="#options" class="hash-link" aria-label="Direct link to Options" title="Direct link to Options" translate="no">​</a></h2>
<table><thead><tr><th>Option</th><th>Default</th><th>Description</th></tr></thead><tbody><tr><td><code>--output-file (-o)  &lt;output-file&gt;</code></td><td><code>./rootapp-&lt;manifest.version&gt;.pkg</code> (dots in version replaced by hyphens)</td><td>Output file for the generated package. Example: if <code>manifest.version</code> is <code>1.2.3</code>, the output is <code>./rootapp-1-2-3.pkg</code>. You can override this by specifying a custom file or directory path.</td></tr><tr><td><code>--project-folder (-p) &lt;folder&gt;</code></td><td><code>./</code> (current directory)</td><td>Path to the directory containing <code>root-manifest.json</code>.</td></tr></tbody></table>
